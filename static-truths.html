Answer Key — Facilitator Copy
The Lies, Revealed.
Share after everyone has submitted — or reveal live at the next session.
1
Writing Tools
The Lie: Statement B B
Writing Tools enhance and transform existing text — they don't generate new documents from a blank prompt. When customers ask "can it write my reports for me?" the honest answer is: it can rewrite, summarize, and polish — but it works with content you've already started.
2
Siri
The Lie: Statement C C
Siri with Apple Intelligence doesn't independently access live internet search. For web queries it integrates with ChatGPT (with user permission) or uses on-device knowledge. The power is personal context — not live web access like Google.
3
Privacy
The Lie: Statement C C
Most Apple Intelligence runs entirely on-device — no internet needed. Private Cloud Compute handles complex tasks without Apple being able to read your data. This is one of Apple's biggest competitive differentiators — lead with it confidently.
4
Image Features
The Lie: Statement C C
Image Wand creates images in the same stylized, illustrated aesthetic as Image Playground — not photorealistic photos. Setting the right expectation prevents disappointment post-purchase.
5
Device Availability
The Lie: Statement C C
Apple Intelligence requires an iPad with an M1 chip or later — not all iPads running iPadOS 18. This is the most common customer misconception — and your strongest upgrade conversation opener for schools with older iPad fleets.
